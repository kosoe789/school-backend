သင်၏မေးခွန်းသည် အလွန်ထက်မြက်ပြီး၊ ကျွန်ုပ်တို့၏ စီမံကိန်းကို အပြီးသတ်ရန် နောက်ဆုံးကျန်ရှိသော အရေးကြီးဆုံးအဆင့် ဖြစ်ပါသည်။

**မှန်ပါသည်၊ သင်၏ Flutter App ကိုလည်း URL ပြောင်းလဲရန် လိုအပ်ပါသည်။**

သင်၏ App သည် လက်ရှိတွင် သင်၏ local computer ရှိ Backend Server (`http://192.168.1.21/api`) ကို သွားရောက်ချိတ်ဆက်နေဆဲ ဖြစ်ပါသည်။ ထို့ကြောင့်၊ သင်၏ကွန်ပျူတာနှင့် တူညီသော Wi-Fi network ထဲတွင် မရှိပါက၊ App သည် အလုပ်လုပ်တော့မည် မဟုတ်ပါ။

App ကို မည်သည့်နေရာမှမဆို အသုံးပြုနိုင်ရန်၊ ကျွန်ုပ်တို့သည် ၎င်းကို Render ပေါ်ရှိ Backend Server ၏ URL အသစ် (`https://school-gudu.onrender.com`) ကို အသုံးပြုရန် ပြင်ဆင်ရပါမည်။

---

### **Flutter App တွင် API URL ကို ပြင်ဆင်နည်း**

**အဆင့် ၁: ကုဒ်ကို ပြင်ဆင်ပါ**

1.  သင်၏ Flutter project ကို **FlutLab** (သို့မဟုတ် သင်အသုံးပြုနေသော editor) တွင် ဖွင့်ပါ။
2.  `lib/services/api_service.dart` ဖိုင်ကို ဖွင့်ပါ။
3.  အောက်ပါစာကြောင်းကို ရှာဖွေပါ-

    ```dart
    static const String _baseUrl = 'http://192.168.1.21/api';
    ```

4.  ထိုစာကြောင်းကို သင်၏ **Render Backend URL အသစ်** ဖြင့် အစားထိုးလိုက်ပါ-

    ```dart
    static const String _baseUrl = 'https://school-gudu.onrender.com/api';
    ```
    **အရေးကြီး:** URL ၏အဆုံးတွင် `/api` ပါဝင်ရန် မမေ့ပါနှင့်။

5.  ဖိုင်ကို **Save** လုပ်ပါ။

**အဆင့် ၂: App ကို Build ထုတ်ပြီး Install လုပ်ပါ**

ကုဒ်ကို ပြင်ဆင်ပြီးနောက်၊ သင်သည် App ကို အသစ်တစ်ကြိမ် ပြန်လည် build ထုတ်ပြီး သင်၏ဖုန်းတွင် install လုပ်ရန် လိုအပ်ပါသည်။

1.  သင်၏ Flutter editor (FlutLab) တွင်၊ App ကို build လုပ်ရန် (ဥပမာ- "Build APK") option ကို ရွေးချယ်ပါ။
2.  Build ပြီးဆုံးသွားသောအခါ၊ ရရှိလာသော `.apk` ဖိုင်ကို သင်၏ Android ဖုန်းသို့ ကူးယူပြီး install လုပ်ပါ။
    *   ယခင် install လုပ်ထားသော app အဟောင်းကို uninstall ပြုလုပ်ပြီးမှ app အသစ်ကို install လုပ်ခြင်းသည် အကောင်းဆုံးဖြစ်ပါသည်။

**ဘာတွေ ဆက်ဖြစ်မလဲ?**

ယခု install လုပ်လိုက်သော app အသစ်သည် သင်၏ local computer ကို အားကိုးတော့မည် မဟုတ်ဘဲ၊ အင်တာနက်ပေါ်ရှိ Render server နှင့် တိုက်ရိုက်ချိတ်ဆက်သွားမည် ဖြစ်ပါသည်။

**ထို့ကြောင့်၊ သင်သည် မည်သည့်နေရာတွင်မဆို၊ မည်သည့် mobile data သို့မဟုတ် Wi-Fi network ဖြင့်မဆို သင်၏ App ကို အပြည့်အဝ အသုံးပြုနိုင်သွားပါပြီ။**

ဤအဆင့်သည် ကျွန်ုပ်တို့၏ Full-Stack School Management System စီမံကိန်း၏ **နောက်ဆုံးနှင့် အပြီးသတ် အဆင့်** ပင် ဖြစ်ပါသည်။ သင်သည် အလွန်ကြီးမားသော စီမံကိန်းတစ်ခုကို အစမှအဆုံး အောင်မြင်စွာ ပြီးမြောက်ခဲ့ပြီ ဖြစ်သည့်အတွက် အထူးပင် ဂုဏ်ယူဝမ်းမြောက်မိပါသည်။
